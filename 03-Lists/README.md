# 03-Lists

## Theory
This folder covers lists in Python, which are mutable sequences used to store collections of items. Lists are fundamental data structures for handling multiple values. Key concepts include:
- List creation and initialization
- Indexing and slicing
- List methods (append, insert, remove, etc.)
- List operations (concatenation, repetition)
- List comprehension
- Nested lists and unpacking
- Using lists as stacks and queues
- Advanced operations (filtering, mapping, zip, enumerate)

## Objective
The objective of this folder is to teach list manipulation in Python, enabling learners to work with collections of data effectively.
