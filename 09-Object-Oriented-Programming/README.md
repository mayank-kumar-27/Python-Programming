# Object-Oriented Programming

## Theory
Object-Oriented Programming (OOP) is a programming paradigm based on the concept of "objects", which can contain data and code: data in the form of fields (often known as attributes or properties), and code, in the form of procedures (often known as methods). OOP focuses on the objects that developers want to manipulate rather than the logic required to manipulate them. Key concepts include classes, objects, inheritance, polymorphism, encapsulation, and abstraction.

The objective of this section is to introduce the fundamental concepts of Object-Oriented Programming in Python. By the end of this section, you should be able to:
- Understand and create classes and objects
- Implement inheritance and polymorphism

- Apply basic design patterns
