# 01-Basics

## Theory
This folder covers the fundamental concepts of Python programming, including data types, variables, constants, operators, and basic input/output operations. Understanding these basics is essential as they form the building blocks for all Python programs. Key concepts include:
- Data types (int, float, str, bool)
- Variables and their declaration
- Constants and naming conventions
- Basic I/O functions (input, print)
- Operators (arithmetic, comparison, logical, assignment, bitwise)
- Comments and program structure

## Objective
The objective of this folder is to provide a solid foundation in Python programming basics, enabling learners to write simple programs that can read input, perform basic operations, and display output.
